# Taipei Tech Refined #
本軟體接受GPLv3授權條款並修改自 **TaipeiTechApp https://github.com/AlanChiou/TaipeiTechApp**

## 介紹 ##
在原程式的架構下，修正既有 Bug 、移除 Google Analytics，並做出下列修改：  
  - *暫時*移除活動報報頁面  
  - 移除捐贈頁面  
  - 追加已撤選課堂偵測功能  
  - 新增英文 (UI + 課表) 、日文介面 (UI)  
  - 新增問題回報頁面
